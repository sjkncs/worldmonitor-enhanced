## 🚀 Quick Start

### Web App (Instant)
Visit [worldmonitor.app](https://worldmonitor.app) — no installation required

### Desktop App (Native Performance)
Download for [Windows](https://worldmonitor.app/api/download?platform=windows-exe) | [macOS ARM](https://worldmonitor.app/api/download?platform=macos-arm64) | [macOS Intel](https://worldmonitor.app/api/download?platform=macos-x64) | [Linux](https://worldmonitor.app/api/download?platform=linux-appimage)

### Self-Hosted (Developer)

```bash
# Clone repository
git clone https://github.com/koala73/worldmonitor.git
cd worldmonitor

# Install dependencies
npm install --legacy-peer-deps --ignore-scripts

# Start development server
npm run dev

# Access at http://localhost:3000
```

### Test New Features

```bash
# Visit test page
http://localhost:3000/test-new-features.html

# Or use browser console
const military = await import('/src/services/military-tracking.ts')
const aircraft = await military.fetchOpenSkyMilitary()
console.log(`Found ${aircraft.length} aircraft`)

const energy = await import('/src/services/energy-intelligence.ts')
const prices = await energy.fetchEnergyPricesYahoo()
console.log('Energy prices:', prices)
```

## 🤝 Contributing

We welcome contributions! Please see our [Contributing Guide](./CONTRIBUTING.md).

### Development Setup

```bash
# Install dependencies
npm install --legacy-peer-deps

# Run development server
npm run dev

# Run tests
npm run test:e2e

# Type check
npm run typecheck
```

1. Create service in `src/services/`
2. Export in `src/services/index.ts`
3. Create Panel component in `src/components/`
